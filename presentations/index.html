การเชื่อมโยงกิจ IO กับ DECISIVE POINTS (STEP 4) — DP ↔ IO LINKAGE

นำ Decisive Points (DP ๑–๑๔) จากขั้นที่ ๔ «จัดทำสภาวะที่ต้องการทางการรบ» มาเชื่อมโยงไปยังกล่องกิจ IO ทางด้านขวา แยกเป็น ๒ ผัง — กิจที่กระทำเพื่อ ป้องกันฝ่ายเรา (Defensive) และกิจที่กระทำ ต่อ WL (Offensive)  ·  ชี้ที่ DP หรือกล่อง IO เพื่อเน้นเส้นเชื่อมโยง

เส้นผังป้องกันฝ่ายเรา เส้นผังต่อ WL ★ DP13 = Decisive Point ด้าน IO (ความพยายามหลัก) DP ที่จาง = ไม่มีกิจหลักในผังด้านนั้น
🛡 กิจ IO — ด้านป้องกันฝ่ายเรา (Defensive / Protect)
⚔ กิจ IO — ด้านกระทำต่อ WL (Offensive)

★ DP13 (ผลกระทบพลเรือนจำกัด + นานาชาติยอมรับ) คือ Decisive Point ด้าน IO — กิจ PA/StratComm, Counter-JORINO และ CMO/KLE รวมศูนย์ที่นี่ ส่วน DP อื่นเป็นกิจสนับสนุน  ·  หมายเหตุ: ช่องทาง PA (ข้อเท็จจริงต่อสาธารณะ) ต้องแยกจาก PSYOP/MILDEC (ต่อข้าศึก) เพื่อรักษาความน่าเชื่อถือ — ตัวเลข «นานาชาติยอมรับ» เป็นค่าสมมุติเชิงสถานการณ์ ใช้เป็นเป้าวัด (MOE) มิใช่ข้อเท็จจริงอ้างอิง
